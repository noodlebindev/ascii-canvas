**Mode:** Unicode (default)

---

## Why this works

- Six rungs map to the authentic progression curve for Spanish verb conjugation: foundations → core tenses → application → subjunctive → compound → literary/dialectal edge cases. Each rung genuinely requires the one below.
- Three bullets per rung (instead of the skeleton's two) add enough sub-skill detail to make each level actionable — a learner knows exactly what to practice, not just a vague level name.
- Box width is 74 inner chars (76-wide inner count to the `│`), keeping each rung well within the 78-char inner budget at 80-col format width.
